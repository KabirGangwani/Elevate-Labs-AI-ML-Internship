# Elevate Labs AI/ML Internship

This repository contains the tasks completed during the Elevate Labs AI & Machine Learning Internship.

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab
- KaggleHub

## Tasks Completed

### Task 1: Titanic Data Preprocessing
- Loaded and explored the Titanic dataset.
- Handled missing values.
- Encoded categorical variables.
- Performed feature scaling and preprocessing.
- Prepared the dataset for machine learning.

### Task 2: Exploratory Data Analysis (EDA)
- Performed statistical analysis of the dataset.
- Visualized distributions and relationships between features.
- Identified trends, patterns, and correlations.
- Generated insights using various plots and charts.

### Task 3: Housing Price Prediction using Linear Regression
- Downloaded and loaded the Housing Price Prediction dataset.
- Performed data preprocessing and feature encoding.
- Split the dataset into training and testing sets.
- Implemented Simple and Multiple Linear Regression.
- Evaluated model performance using:
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)
  - R² Score
- Visualized regression results and interpreted model coefficients.
